Personal Viewing Data Analyzer

Python Matplotlib Data Analysis Data Visualization

A Python-based tool that analyzes Netflix viewing activity data to visualize watch patterns, time spent on different shows, and viewing habits over time.

The Story Behind PVDA

Recently, I was going through my Netflix's "My Account" page and realized that you could download your profile's viewing activity in a CSV format. I immediately thought it would be pretty cool to visualize my Netflix usage patterns.

Lately, I have been practicing my Python skills, and this seemed like a perfect opportunity to use Matplotlib libraries to create meaningful visualizations from real-world data.

Key Features

  • Parse and process Netflix viewing history CSV files
  • Calculate total watch time per show/movie
  • Generate visual charts showing viewing patterns
  • Analyze viewing habits by time of day and day of week
  • Create insightful visualizations using Matplotlib

Technologies Used

Python Pandas Matplotlib CSV Processing
